pglevy / web-tools

Small tools. Everyday use.

A collection of standalone HTML utilities. No build step, no framework — open a file and it works. Inspired by Simon Willison's approach to single-file tools.

Single file

Each tool is one HTML file with inline CSS and JS. Copy, paste, host anywhere.

No build step

No React, no npm, no bundler. Dependencies load from a CDN when needed.

Stays small

A few hundred lines. Easy to read, easy to modify, easy to hand off to an LLM.

An interactive decision tree to help you choose the right AI context mechanism — steering files, skills, hooks, MCP servers, and more.
